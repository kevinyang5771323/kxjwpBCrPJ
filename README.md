# 个人理财管理系统 python352

## 项目概述

个人理财管理系统是基于Python的Django框架开发的一个记账工具，旨在帮助用户高效管理个人财务，进行日常收支的记录和分析。

## 技术栈

- **后端语言**：Python 3.x
- **框架**：Django
- **数据库**：MySQL

## 功能模块

系统主要包括以下功能模块：

- 用户管理：注册、登录、个人信息管理
- 收入管理：添加收入、收入记录查看、收入分类统计
- 支出管理：添加支出、支出记录查看、支出分类统计
- 财务报表：月度报表、年度报表、自定义时间范围报表
- 预算管理：预算设置、预算执行跟踪

## 系统角色

- **普通用户**：可以进行日常的记账操作，查看财务报表，设置和管理个人预算。
- **管理员**：负责用户管理和系统维护。

## 运行环境

- **开发环境**：PyCharm
- **操作系统**：兼容Windows、Linux和macOS
- **依赖管理**：使用pip进行Python依赖包管理

## 部署步骤

1. 环境准备：安装Python、MySQL和PyCharm。
2. 克隆或下载项目代码。
3. 使用pip安装项目依赖。
4. 配置数据库：创建数据库和用户，并设置权限。
5. 运行迁移命令，以应用数据库迁移。
6. 启动Django应用。

## 核心亮点

- **灵活的记账功能**：用户可以根据需求自定义收入和支出的分类。
- **强大的报表系统**：支持多种时间范围的财务报表生成，帮助用户清晰了解财务状况。
- **预算管理**：用户可以设定预算，系统帮助跟踪预算执行情况。
- **跨平台运行**：支持主流操作系统，用户可以在不同的设备上管理财务。
- **易于部署**：遵循Python和Django的标准化部署流程，简化部署步骤。
